# Rock-Paper-Scissors (AI Opponent)

This is a simple Rock-Paper-Scissors game built in Python where the user plays against the computer.

## Objective
The goal of this project is to create a basic game using control flow, randomization, and simple logic.

## Tools Used
- Python
- random module

## How the AI Works
- In the beginning, the AI selects moves randomly.
- After a few rounds, it tracks the user's move patterns.
- It predicts the most frequent move and tries to counter it.
- It also uses some randomness to avoid being predictable.

## How to Run
1. Make sure Python is installed.
2. Run the file:
   python filename.py
3. Enter your move: rock, paper, or scissors.
4. Type "quit" to exit the game.

## Features
- Score tracking (user, AI, ties)
- Input validation
- Basic AI behavior

## Learning Outcomes
- Understanding of if/else conditions
- Use of loops
- Basic implementation of game logic
- Simple AI decision making
